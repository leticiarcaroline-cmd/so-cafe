# So-Cafe
Site institucional de uma cafeteria especial, desenvolvido como projeto de aprendizado.

# Tecnologias utilizadas 
- HTML5
- CSS3
- Javascript
- Fetch API
- JSON 

# Funcionalidades 

- Menu do cardápio carregando dinamicamente via Fecht API
- Galeria de fotos 
- Navegação suave entre seções
- Layout responsivo 

# O que eu aprendi 

- Manipulação do DOM com JavaScript 
- Uso de array de objetos com forEach
- Consumo de dados externos com Featch API
- Versionamento com Git e GitHub
